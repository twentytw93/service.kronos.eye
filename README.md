# Kronos Eye

Kronos Eye is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It tracks lunar cycles and planetary alignments, providing real-time celestial status updates directly inside Kodi.

---

## Features
- **Full Moon Tracking** – Displays current lunar phase and upcoming full moon dates. 
- **Saturn Cycle Monitoring** – Tracks Saturn positioning and timing events. 
- **Celestial Alerts** – Sends visual notifications during major astronomical shifts. 
- **Background Observer Engine** – Runs efficiently with minimal system impact. 
- **Accurate Time Sync** – Uses system time for precise celestial calculation. 
- **Refined Visual Output** – Minimal and clean presentation inside the Kodi interface.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Arctic Zephyr Reloaded skin 
- Python 3 (bundled with Kodi)
